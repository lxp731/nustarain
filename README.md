### How to run

Clone the repository:

```bash
git clone https://github.com/lxp731/nustarain.git
```

Install dependencies and run:

```bash
npm install
hexo cl && hexo g && hexo s
```

> Ref: [Install Hexo](https://hexo.io/docs/#Install-Hexo)
